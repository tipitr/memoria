# MOMENTUM GAME V3
## Internal Multi-Agent Design Brainstorm

---

# AGENT 1 — Neuropsychology Agent
## “Will this actually help ADHD brains?”

### HARD TRUTH
ADHD users don’t consistently lack motivation.

They lack:
- activation stability
- transition control
- emotional regulation under friction

So:
> the game should optimize state transition,
NOT productivity output.

---

## Biggest Risk Detected
If every interaction becomes:
- “do task”
- “reward”
- “next task”

the brain adapts quickly and dopamine collapses.

ADHD brains habituate FAST.

---

## Recommendation:
The reward should NOT be:
# task completion

It should be:
# state improvement

Examples:
- calmer room
- clearer soundscape
- reduced fog
- stable lighting
- companion relaxes

This creates:
> emotional regulation feedback

instead of:
> productivity pressure

---

## Critical Insight
ADHD users are hypersensitive to:
- perceived judgment
- invisible expectations
- broken streak shame

Therefore:

### REMOVE:
- streaks
- percentages
- “daily goals”
- red notifications
- unfinished warnings

---

## ADD:
# “Recovery Design”

When user disappears for 10 days:
NOT:
> “You missed 37 quests”

BUT:
> “The room stayed warm for you.”

This matters psychologically more than people think.

---

# AGENT 2 — Game Director Agent
## “Is this actually a game?”

### HARD TRUTH
Right now this risks becoming:
> emotionally intelligent Todoist

Needs:
- mystery
- discovery
- identity
- progression fantasy

---

# Recommendation:
## Turn the brain into an explorable world

NOT just:
room simulator

But:
- memory corridors
- focus gardens
- overwhelm storms
- abandoned thoughts
- unfinished idea forests

Now the player explores:
> their cognitive landscape

---

# BIG MECHANIC IDEA
# “Attention Ecology”

Every action affects ecosystem balance.

Example:

Too many unfinished tasks:
→ crows appear
→ static noise increases
→ pathways clutter

Small consistent actions:
→ plants regrow
→ light returns
→ new areas unlock

This creates:
# systemic gameplay

instead of linear task loop.

---

# IMPORTANT
Do NOT make player “fight ADHD”

That framing is wrong.

Instead:
> coexist with cognitive weather

This emotional framing changes everything.

---

# PROGRESSION SYSTEM

NOT:
- levels
- XP grind
- battle pass

BUT:
# “Self Understanding Unlocks”

Player gradually discovers:
- what environments help them
- what momentum patterns work
- what emotional states create friction

This becomes:
# meta progression

---

# AGENT 3 — Systems Designer Agent
## “Can this actually be built?”

### HARD TRUTH
A fully adaptive AI companion world simulation:
- too expensive
- too inconsistent
- impossible for MVP

Need:
# illusion of intelligence

NOT true AGI.

---

# RECOMMENDED TECH ARCHITECTURE

## Layer 1 — State Engine
Simple variables:
- energy
- overwhelm
- focus
- momentum
- noise

This drives world state.

---

## Layer 2 — Task Classification
LLM-assisted.

Classify:
- admin
- creative
- emotional
- cleaning
- ambiguous

NOT full autonomous planning.

---

## Layer 3 — Rule-Based Tiny Quest Engine

THIS is core.

Example rules:
IF:
- anxious + creative task

THEN:
- generate mechanical first step

Example:
“write title only”

Reliable.
Predictable.
Cheap.

---

## Layer 4 — Companion Behavior Engine

Not conversational AI.

Instead:
- animation states
- small phrases
- memory tags
- behavioral adaptation

This scales.

---

# MOST IMPORTANT MVP DECISION

DO NOT BUILD:
- open world
- multiplayer
- complex AI

MVP should be:

# ONE ROOM
with:
- emotional weather
- tiny quest engine
- attention spirit
- world transformation

That’s enough.

---

# AGENT 4 — Ethical Design Agent
## “Will this accidentally become addictive?”

### BIG WARNING
ADHD users are vulnerable to:
- compulsive loops
- optimization addiction
- avoidance through systems

Danger:
Player becomes addicted to:
> organizing life instead of living life

---

# Recommendation:
## Design “exit velocity”

Game should encourage:
- leaving app
- entering real life
- returning naturally

NOT endless engagement.

---

# Mechanic Suggestion:
## “World Sleeps While You Act”

The best progression happens:
AFTER player leaves app and does real action.

Brilliant psychologically.

Example:
- player starts momentum sprint
- leaves app
- returns 20 mins later
- room evolved

This avoids:
> productivity theater
